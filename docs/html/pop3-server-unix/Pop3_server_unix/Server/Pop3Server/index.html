val start : hostname:string -> maildrop:string -> stop:unit Lwt.t -> unit Lwt.t

Starts a TCP server listening on port 110 to serve POP3 client connections.

There is no limit to the number of concurrent connections and connections do not timeout.

parameter hostname

is the hostname of the server used in APOP digests.

parameter maildrop

is the path to the maildrop on disk.

parameter stop

is the lightweight thread which if realised can terminate the server.

returns

unit lightweight thread running the server.
